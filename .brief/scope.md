# Scope Proiect

## Pagini

| Pagina | Prioritate | Descriere |
|--------|-----------|-----------|
| Home | Must-have | Pagina principala cu hero, prezentare servicii, track record/stats, portofoliu preview, testimoniale, CTA-uri, newsletter |
| Despre noi | Must-have | Povestea AceAgency, echipa, valori, misiune, viziune, parteneriate |
| Servicii | Must-have | Pagina generala servicii + sub-pagini per serviciu/divizie (AceWeb, AceAds, AceAI, AceMedia) |
| Contact | Must-have | Formular contact, Google Maps, informatii contact, Calendly booking |
| Portofoliu / Studii de caz | V2 | Showcase lucrari, studii de caz cu rezultate masurabile |
| Blog | V2 | Articole, noutati, insights industrie, ghiduri |
| Echipa | V2 | Pagina dedicata membrilor echipei cu roluri si competente |

**Total**: 7 pagini (4 must-have, 3 V2)

## Functionalitati

| Functionalitate | Prioritate | Note |
|----------------|-----------|------|
| Formular contact | Must-have | Cu validare Zod, honeypot anti-spam, email via Resend |
| Newsletter signup | Must-have | Colectare email-uri, integrare Resend |
| Social media links | Must-have | LinkedIn, Instagram, Facebook, etc. |
| Multi-language (RO/EN) | Must-have | next-intl cu switch de limba |
| Analytics | Must-have | Vercel Analytics + Google Analytics 4 + GTM |
| Cookie consent + GDPR | Must-have | Banner cookies, Privacy Policy, Terms |
| Animatii avansate | Must-have | Framer Motion / GSAP - scroll animations, parallax, reveals (stil addifico.com) |
| Blog / CMS | V2 | Sistem de administrare continut - se adauga la final |

## Excluderi

- E-commerce / plati online
- User authentication / conturi utilizator

## Complexitate

- **Nivel**: Medium
- **Pagini**: 7
- **Functionalitati**: 8
- **Integratii**: 4 (Resend, Google Maps, GA4, Calendly)
